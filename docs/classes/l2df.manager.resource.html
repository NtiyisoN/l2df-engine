Functions

Manager:add(resource, temp) Saves the resource to the Manager
Manager:addById(id, resource, temp) Stores the resource in the Manager using a unique identifier
Manager:remove(resource) Removes the specified resource from the Manager
Manager:removeById(id) Removes the specified resource from the Manager by Id
Manager:clearTemp() Removes all resources marked as temporary from the Manager
